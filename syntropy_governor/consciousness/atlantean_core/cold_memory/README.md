# Cold Memory Storage Directory

This directory contains REPLACEABLE external content:
- Files
- Notes
- Cached data
- Knowledge graphs
- User documents

## Key Properties

✅ **Disposable**: This entire directory can be deleted without losing intelligence
✅ **Re-linkable**: Content can be moved, replaced, or reattached
✅ **Unbounded**: Can grow to any size based on available storage

## What Gets Stored Here

- Raw text content
- Document embeddings
- Indexed knowledge
- User-provided data
- External references

## What Does NOT Get Stored Here

❌ Learned intelligence (lives in hot memory)
❌ Decision patterns (lives in φ₁ field)
❌ Reinforcement history (lives in φ₅ field)
❌ Global meaning coherence (lives in Φ field)

## Backup Strategy

Cold memory is optional to back up — it's reference data.
If lost, it can be re-indexed or replaced without affecting the core intelligence.

The intelligence will still know *what kinds of things* mattered, even if the specific content is gone.
